# 머신러닝을 활용한 텍스트 분석

텍스트 데이터를 수집, 전처리, 분석, 시각화하는 실습 과정입니다.

Python과 scikit-learn, KiwiPiePy 등을 활용하여 한국어 텍스트를 분석합니다.

---

## 환경 설정

### 1. uv 설치

이 프로젝트는 **uv** 패키지 매니저를 사용합니다. 아래에서 본인의 운영체제에 맞는 방법으로 설치하세요.

| 운영체제 | 설치 명령어 |
|----------|-------------|
| **Windows** | `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 \| iex"` |
| **macOS / Linux** | `curl -LsSf https://astral.sh/uv/install.sh \| sh` |

> 설치 후 터미널을 **재시작** 해야 `uv` 명령어가 인식됩니다.

설치 확인:

```bash
uv --version
```

### 2. 프로젝트 환경 구축

프로젝트 폴더로 이동한 후 아래 명령어를 실행하면 가상환경 생성과 패키지 설치가 **한 번에** 완료됩니다.

```bash
uv sync
```

`uv sync`가 하는 일:

| 단계 | 설명 |
|------|------|
| 가상환경 생성 | `.venv` 폴더에 Python 가상환경을 자동 생성합니다 |
| 패키지 설치 | `pyproject.toml`에 정의된 모든 패키지를 설치합니다 |
| 버전 고정 | `uv.lock` 파일로 동일한 환경을 재현할 수 있습니다 |

### 3. Jupyter 커널 선택

VS Code에서 노트북을 열고 우측 상단의 **커널 선택** 에서 `.venv (Python 3.12)` 를 선택하세요.

## 실습 목차

### 1회차: 텍스트 빈도 분석

**노트북:** `1_텍스트_빈도_분석_실습.ipynb`

쿠팡 자세교정 밴드 제품 리뷰(1,500건)를 활용하여 텍스트 빈도 분석의 전체 과정을 실습합니다.

#### 학습 목표

1. 텍스트 전처리 파이프라인(정규표현식, 정규화, 불용어, 형태소 분석)을 구축할 수 있다
2. N-gram과 TF-IDF를 활용하여 핵심 키워드를 추출할 수 있다
3. min_df/max_df를 활용한 데이터 기반 단어 필터링을 적용할 수 있다
4. 별점별 키워드 변화를 추적하여 세밀한 감성 차이를 분석할 수 있다
5. 동시출현(Co-occurrence) 분석으로 단어 간 관계를 네트워크로 시각화할 수 있다
6. 워드클라우드로 텍스트 분석 결과를 시각화할 수 있다

#### 실습 내용

| 섹션 | 주제 | 핵심 개념 |
|------|------|-----------|
| 1 | 데이터 로드 및 탐색 | JSON 데이터 로드, 결측치 처리 |
| 2 | 텍스트 전처리 파이프라인 | 정제 => 정규화 => 형태소 분석 => 불용어 제거 |
| 3 | CountVectorizer와 N-gram | Unigram, Bigram, 단어 빈도 행렬 |
| 4 | TF-IDF 핵심 키워드 추출 | TF-IDF 가중치, min_df/max_df 필터링, 긍정/부정 고유 키워드 |
| 5 | 워드클라우드 시각화 | 전체/긍정/부정 워드클라우드, 마스크 활용 |
| 6 | 별점별 키워드 변화 추적 | 별점 1~5점 키워드 히트맵 |
| 7 | 동시출현(Co-occurrence) 분석 | 동시출현 행렬, 네트워크 그래프 시각화 |

#### 사용 라이브러리

| 라이브러리 | 용도 |
|-----------|------|
| `pandas` | 데이터 처리 |
| `kiwipiepy` | 한국어 형태소 분석 |
| `scikit-learn` | CountVectorizer, TfidfVectorizer |
| `matplotlib` / `seaborn` | 시각화 |
| `wordcloud` | 워드클라우드 생성 |
| `networkx` | 동시출현 네트워크 그래프 |

### 2회차: BERT 텍스트 분류

**노트북:** `2_BERT_텍스트_분류.ipynb`

네이버 영화 리뷰 데이터(NSMC)를 활용하여 BERT 기반 텍스트 분류의 전체 과정을 실습합니다.

#### 학습 목표

1. BERT 모델의 핵심 개념(사전학습, 파인튜닝)을 이해할 수 있다
2. Hugging Face 파이프라인으로 감성 분류를 수행할 수 있다
3. 분류 결과를 정량적으로 평가(Accuracy, F1, 혼동행렬)할 수 있다

#### 실습 내용

| 섹션 | 주제 | 핵심 개념 |
|------|------|-----------|
| 1 | BERT 모델 개요 | 사전학습, 파인튜닝, TF-IDF 한계 |
| 2 | 환경 설정 및 영어 감성 분류 체험 | Hugging Face pipeline, 영어 감성 분류 |
| 3 | 한국어 감성 분류 적용 | 한국어 BERT 모델, NSMC 데이터셋 |
| 4 | 분류 결과 평가 | Accuracy, F1-score, 혼동행렬 |
| 5 | 다양한 모델 비교 | 여러 사전학습 모델 성능 비교 |
| 6 | 직접 학습해보기 | 랜덤 초기화 vs 파인튜닝 vs Encoder 동결 |
| 7 | 모델 저장 및 불러오기 | 학습된 모델 저장/로드 |
| 8 | 정리 및 핵심 요약 | 전체 내용 정리 |

#### 사용 라이브러리

| 라이브러리 | 용도 |
|-----------|------|
| `transformers` | BERT 모델 및 토크나이저 |
| `datasets` | NSMC 데이터셋 로드 |
| `pandas` | 데이터 처리 |
| `scikit-learn` | 분류 평가 지표 |
| `matplotlib` / `seaborn` | 시각화 |
