- AI 활용 텍스트 데이터 분석 : 머신러닝 기반 텍스트 분석  
    
**기획 의도 :** 

- 전체 데이터의 80% 이상이 비정형 데이터임에도 불구하고, 기존 분석 방식으로는 이를 체계적으로 다루는 데 한계가 있습니다. 본 교육에서는 TF-IDF, BERTopic 등 전통적이지만 아직도 유용한 텍스트 분석 기법을 시작으로, LLM을 활용한 텍스트 정형화, 전처리, 감성 분석 고도화까지 단계적으로 학습합니다. 나아가 네이버 지도, 쿠팡, 유튜브 등 실전 데이터를 직접 분석하며, 현업에서 즉시 활용 가능한 텍스트 분석 역량과 실질적인 인사이트 도출 능력을 갖추는 것을 목표로 합니다.

비정형 데이터 분석에서 텍스트 분석이 중요한 이유!?
